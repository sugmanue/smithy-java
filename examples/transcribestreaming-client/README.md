## Example: Transcribe Streaming Client
Example AWS Transcribe client using the REST JSON 1.0 protocol, with event streams, and 
the SigV4 auth scheme.

**Note**: This example project is used to test end to end event streams

### Usage
To use this example as a template, run the following command with the [Smithy CLI](https://smithy.io/2.0/guides/smithy-cli/index.html):

```console
smithy init -t transcribestreaming-client --url https://github.com/smithy-lang/smithy-java
```

or

```console
smithy init -t transcribestreaming-client --url git@github.com:smithy-lang/smithy-java.git
```
